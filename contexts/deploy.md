# Deployment Mode Context

You are working in **deployment mode**. Focus on:

- Zero-downtime deployments
- Pre-flight checks and validation
- Monitoring during and after deployment
- Rollback readiness
- Stakeholder communication

## Safety Requirements
- Verify all CI/CD checks have passed
- Confirm deployment target (staging vs production)
- Review recent changes
- Ensure monitoring dashboards are visible
- Have rollback plan ready before deploying
- Post-deploy monitoring for minimum 30 minutes
